ABOUT

We build tools that make AI more efficient, accessible, and useful. Our research ships as products.

Every tool we release started as a problem we hit in our own work. Model context too expensive? We built a compression model. Need to run five agents at once? We built a multiplexer. The research is always grounded in practice.

We run inference on consumer hardware and home servers. We build for Apple Silicon. We open source everything. Not because it's trendy, but because the fastest way to advance applied AI is to let people build on your work.

Based in Pittsburgh.
